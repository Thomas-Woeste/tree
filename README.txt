Contents:
trees.cpp- This is the main file of the program. It contains the implementation of the binary search tree
           with the relevant functions.
README.txt- Implementation notes on the project
Makefile- This file tells the computer how to compile and run the program.

Executing the program:
To run the program download the trees.cpp file as well as the included Makefile. Type make into the terminal
and the program will download and execute the needed files. Press q to exit the less command where the
program outputs its results.

References:
The basis of building the binary search trees was based on Dr. Finkel's online notes given to student. The
functions were then changed to c++ and to meet the assignments needs.
